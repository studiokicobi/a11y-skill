# Inspiration

This tool draws inspiration from several public accessibility agent/skill projects.

- Community-Access/accessibility-agents
  - inspiration: specialist coverage, multi-platform packaging
  - not adopted directly: multi-agent team architecture

- zivtech/a11y-meta-skills
  - inspiration: lifecycle structure, critique/test phases
  - not adopted directly: planner/critic bundle model

- AccessLint/claude-marketplace
  - inspiration: contrast analysis, modular helper tools, refactor workflow
  - not adopted directly: marketplace/plugin-first packaging

- salawi45/skill-wcag-accessibility
  - inspiration: explicit modes, compliance framing
  - not adopted directly: legal/reporting-heavy positioning

Key differentiator of this project:
- findings grouped by fix autonomy rather than severity
- single-package auditor with static + runtime + stateful + manual lanes
